Correlation and Causation

What is Correlation

Correlaiton is an association between two sets of variables which are linear. Furthermore, one variable effects the another, which in result is causation. Therefore, it can be determined correlation is used to discover the causations for the variables changing.

3 Methods of Visualizing Relationships

  1. Scatterplots are helpful when comparing linear lines while being able to visualize the data in the graph.
  2. A Matrix has the ability to compare lots of data at once. Correlation can be interpreted using a matrix, but usually is not as helpful as a scatterplot
  3. Parrallel Coordinates Plots, like a matrix, have the ability to compare lots of data and the correlation can be interpreted. Futhermore, coordinates enhance the audience activity with the graph and data where features can be added.
